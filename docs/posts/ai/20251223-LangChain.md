# 浪清，涤非？

## 题记

!!! info "关于该文章的书写进度"

    这篇文章仍在撰写中

由于我在科研中需要使用一个比较简单的 Agentic Workflow，所以我学习了 LangChain；这篇文章是我学习 LangChain 过程中的笔记。

> 沧浪之水清兮，可以濯我衣；沧浪之水浊兮，可以濯我足。——《楚辞·渔夫》

<!-- more -->

聪明的你肯定猜到了，标题中的“浪清”指的是 [LangChain](https://github.com/langchain-ai/langchain)，而“涤非”指的是 [Dify](https://github.com/langgenius/dify)，两者都是相当有名的 Agentic Workflow Platform。

非常令人不爽的一点就是，LangChain 的演进太快了，导致用 AI 写出来的代码基本上都有问题，会用很多 `deprecated feature`。好在有一个 MCP Server 叫做 [Context7](https://context7.com/) 就是针对这个问题设计的。
