# 信号与系统记

昨天晚上在 [CS自学指南](https://csdiy.wiki/)上看到[UCB EE120: Signals and Systems](https://csdiy.wiki/电子基础/signal/)的课程似乎还挺有意思，遂打算寒假自学一下这门课。

<!-- more -->

比起国内讲授的《信号与系统》，这门 EE120 看上去更加注重理论与实践相结合，这也是它看上去有意思的原因。

这门课最精华的部分在 6 个编程作业——用 Python 编程结合学到的信号与系统知识，本篇文章的重点就在此。

[课程主页（In web archive）](https://web.archive.org/web/20200208160206/http://inst.eecs.berkeley.edu/~ee120/fa19/)

这门课现在要 UCB 内网才能访问，但是课程内容可以在[此 Github 仓库](https://github.com/PKUFlyingPig/UCB-EE120)中找到。
